# Real Estate SQL Analysis

## Описание проекта

Проект посвящён анализу рынка жилой недвижимости Санкт-Петербурга и Ленинградской области с использованием PostgreSQL.

В рамках проекта были решены две аналитические ad hoc задачи:

1. Анализ времени активности объявлений о продаже недвижимости.
2. Исследование сезонности публикации и снятия объявлений.

Для повышения качества анализа данные были предварительно очищены от выбросов с использованием перцентилей.

---

## Цель проекта

Определить:

- какие объекты недвижимости продаются быстрее;
- какие характеристики влияют на скорость продажи;
- существуют ли различия между Санкт-Петербургом и Ленинградской областью;
- в какие месяцы наблюдается наибольшая активность продавцов и покупателей;
- как меняются характеристики недвижимости в зависимости от сезона.

## Задача 1. Анализ времени активности объявлений

### Бизнес-задача

Определить, какие категории недвижимости продаются быстрее всего, а какие остаются на рынке длительное время.

### Выполненные шаги

- Очистка данных от выбросов.
- Расчёт стоимости квадратного метра.
- Сегментация объявлений по длительности активности:
  - до 30 дней;
  - 31–90 дней;
  - 91–180 дней;
  - более 180 дней;
  - объявления без информации о снятии.
- Сравнение Санкт-Петербурга и Ленинградской области.

### Исследуемые показатели

- стоимость квадратного метра;
- площадь недвижимости;
- количество комнат;
- количество балконов;
- этажность.

---

## Задача 2. Анализ сезонности

### Бизнес-задача

Выявить месяцы с максимальной активностью продавцов и покупателей на рынке недвижимости.

### Выполненные шаги

- Определение месяца публикации объявления.
- Расчёт месяца снятия объявления с продажи.
- Анализ количества опубликованных объявлений.
- Анализ количества снятых объявлений.
- Исследование динамики:
  - стоимости квадратного метра;
  - площади недвижимости.
